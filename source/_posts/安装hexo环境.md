*npm安装慢*
> 安装nodejs后会自动安装npm,用npm安装node插件，不过通常会很慢，有时候会遇到问题，所以我们可以用淘宝网的镜像安装cnpm
>
```
    执行： npm install cnpm -g --registry = https://registry.npm.taobao.org
``` 
> 这样cnpm也可以和npm一样安装插件了
